# 中尾先生のポートフォリオ

## 説明
公立中学校・社会科教師である中尾和博の自己紹介用Webサイトです。プロフィール、資格・活動、社会科の授業実践、趣味などを紹介しています。

## 主なページ

| ファイル | 内容 |
|----------|------|
| `index.html` | トップページ（プロフィール・詳しい自己紹介・趣味） |

## 使い方

- ローカルで確認する場合：`index.html` をブラウザで開くか、簡易サーバーで配信してください。
- デプロイ：Vercel などにアップロードするとそのまま表示できます。手順は `Vercel連携手順.md` を参照してください。

## 使用技術

- HTML
- CSS（インラインスタイル）
- 静的サイト（Vercel でホスティング可能）
